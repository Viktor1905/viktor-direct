# Landing Page

Статический сайт для директолога. Проект готов к загрузке в GitHub и деплою на Vercel без сборки.

## Структура

- `index.html` - основная страница
- `styles.css` - стили
- `script.js` - логика формы и pop-up
- `note_metrica_dark.png` - изображение для hero
- `vercel.json` - конфигурация Vercel

## Как загрузить на GitHub

1. Создайте новый пустой репозиторий на GitHub.
2. Загрузите в него содержимое этой папки.
3. Проверьте, что в корне репозитория лежат `index.html`, `styles.css`, `script.js`, `note_metrica_dark.png`, `vercel.json`.

## Как задеплоить на Vercel

1. Зайдите в Vercel.
2. Нажмите `Add New -> Project`.
3. Выберите ваш GitHub-репозиторий.
4. Vercel определит проект как статический сайт.
5. Нажмите `Deploy`.

## Важно

- Сайт работает без backend.
- Форма открывает Telegram с готовым сообщением.
- Если потом захотите подключить домен, это делается уже в настройках проекта на Vercel.
